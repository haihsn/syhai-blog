03 — Component

Hairline border + four corner ticks — the sharp-cornered, CAD-inspired stand-in for the reference's rounded shadow card.

Standard frame

Default tick color (ink2). Use for callouts, pull quotes, featured boxes.

Accent frame

One per page max — reserved for the single "featured" element.
